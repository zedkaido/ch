# `ch` AKA. Clipboard History

```
Usage: ch [commands]

Description: A ch(clipboard history) you are in control of.

Commands:
   -a, add
   -c, copy
   -d, delete, del
   -s, select
   -e, edit 
   -clear [(clip)board|(hist)ory]

Dependencies: fzf, fzy, rg
```

## Installation
```
cp ch .local/bin/ch
```
